
# Tiny VAE Colab

A minimal Variational Autoencoder implementation in Google Colab, based on Alexander Van de Kleut's VAE tutorial.

## Overview

This project provides a small and easy-to-understand implementation of a Variational Autoencoder using PyTorch.

It is intended for learning how VAEs work, experimenting with latent spaces, and visualizing reconstructed samples.

## Features

- Lightweight VAE implementation
- PyTorch-based training loop
- Google Colab ready
- Simple model structure
- Easy to modify and extend
